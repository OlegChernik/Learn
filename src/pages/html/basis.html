Visibility - для отображения или скрытия блока
  1. visible - по умолчанию
  2. hidden - скрывает блок, но оставляет месте скрытого элемента

Transition - служит для изменения состояния

  1. transition-duration: определяет время за которое происходит переход
  2. transition-property: указывается одно или несколько свойств в коьорых происходят переходы, поумолчанию - all!
  3. transition-delay: задержка при выполнении перехода свойства
  4. transition-timing-function: задает стиль перехода. EASE - поумолчанию
