Grid * контейнеру необходимо задать свойство display: grid; * для разделения колонок используется значение fr - fraction - фактор гибкости, коэффициент пропорциональности (1fr / 2fr / 5fr ...). * grid делит видимую область на КОЛОНКИ и СТРОКИ (column / row) ! каждая колонка и строка имет порядковый номер (нумерация начинается от ЛЕВОГО верхнего угла) * grid-column-start * grid-column-end * grid-row-start * grid-row-end
