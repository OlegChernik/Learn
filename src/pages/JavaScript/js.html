Методы объекта

Методы - это свойство объекта, значение которого является функция

* Копирование объекта

  1. assign() - является методом объекта "Object.assign({}, obj)" - это свойство глобального объекта, доступ = web && node.js
  2. { ... obj } - оператор 3 точки "Spred" - делит объект на свойства и сразу же собирает все получаемые свойства в новый объект.
  3. JSON.parse(JSON.stringify(obj)) - позволяет создать копию объекта на всех уровнях вложенности

JSON

  1. Json - формат обмена данными в интернете
  2. Запись свойств Json всегда в двойный кавычках!
  3. Передача данных в формате Json между комп. происходит в виде стоки, Json передается в виде строки

* Конвертация JSON файлов

  • parse() - метод объекта JSON, полученные данные в виде строки парсит в объект JS
  • stringify() - метод объекта JSON, форматирует объект JS в строку, для получения формата JSON

Операторы JS

Оператор - это встроинная в JS функция(внутренняя функция), при использовании того или иного оператора, мы запускаем функцию, что приводит к выполнению действий и возвращения результата

Виды Операторов

  • Арифметические
    • +
    • -
    • *
    • /
  • Сравнения
    • >
    • <
    • >=
    • <=
    • ===
    • !=
  • Логические
    • ||
    • &&
    • !

Обработка ошибок

* throw new Error - инструкция для генерирования ошибок

* try / catch - инструкция для ловли ошибок

  • try { помещается часть в которой возможно возникнет ошибка}
  • catch(error) {эта часть выводит инфо ошибки }

Массивы

Создание массива

  • const arrOne = [1,2,3]
  • const arrTwo = new Array (1,2,3)

Функции высшего порядка

  • Push - добавляет элемент в конец массива
  • Pop - удаляет последний элемент в массиве
  • Unshift - добавляет элемент в начало массива
  • Shift - удаляет первый элемент в массиве
  • forEach - переберает все элементы в массиве
  • Map - также ожидает cullback f в качестве арнумента, но Map возвращает новый массив
